How to create new Repository in GitHub using VsCode

  • git init
  • create new repo using github or using vs code goto source control and create repository
  • git remote add origin (url)
  • git add .
  • git commit -m "Commited"
  • git push origin main

How to create new branch

  • git pull -> to back from main branch
  • git checkout - b (new_branch_name) -> to create new branch
  • git push origin (new_branch_name) -> to push your changes into new created branch
